# Study Dashboard (CS230 Lab)

A clean, responsive, and interactive study dashboard built as part of the CS230 Web Development coursework. This project demonstrates the integration of semantic HTML, advanced CSS layouts (Flexbox/Media Queries), and functional JavaScript DOM manipulation.

## 🚀 Features

- **Dynamic Greeting:** Personalizes the dashboard using JavaScript based on the user's name.
- **Session Tracker:** An interactive counter to log and reset daily study sessions.
- **Goal Management:** A dynamically generated list of tasks pulled from JavaScript arrays and rendered to the DOM.
- **Responsive Design:** A mobile-friendly layout that shifts from a three-column desktop view to a single-column mobile view.
- **Modern UI:** Features CSS hover effects, scale transformations, and a clean card-based aesthetic.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties, Flexbox, transitions, and Media Queries.
- **JavaScript (Vanilla):** Event listeners, loops, and DOM selection/manipulation.

## 📸 Preview

* **Desktop:** Cards are arranged horizontally with interactive hover states.
* **Mobile:** Elements stack vertically, and profile images are hidden for a cleaner UX on small screens.

## 📂 Project Structure

- `index.html`: The main structure of the dashboard.
- `style.css`: The styling logic including responsive breakpoints.
- `script.js`: The interactivity logic for the session counter and goals list.
